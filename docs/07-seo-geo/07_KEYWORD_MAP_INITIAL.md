## Рекламная логика

На старте не смешивать все в одну кампанию.

Лучше:

1. Отдельная кампания “ремонт квартир”.
2. Отдельная кампания “отделочные работы”.
3. Отдельные группы по городам.
4. Отдельные объявления под “бесплатный замер + смета + контроль”.
5. Отдельный бюджет на тест каждого города.

## Первичные гипотезы

- Пятигорск: больше спроса, выше конкуренция.
- Кисловодск: хороший потенциал по ремонту недвижимости под проживание/аренду/инвестиции.
- Ессентуки: перспективно по новостройкам и частному сектору.
- Железноводск: меньше спроса, но может быть дешевле заявка.
