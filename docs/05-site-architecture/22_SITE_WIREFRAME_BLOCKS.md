# Stage 3.5 — Site Wireframe Blocks

This document defines the block structure for the MVP landing page.

## 1. Header
Elements:
- logo / project name placeholder;
- navigation: Услуги, Как работаем, Калькулятор, Контакты;
- phone;
- WhatsApp / Telegram CTA;
- mobile sticky contact button.

## 2. First Screen / Hero

Required content:
- ремонт и отделочные работы в Ессентуках;
- бесплатный замер;
- понятная смета;
- контроль качества;
- CTA button.

Recommended headline:
> Ремонт и отделочные работы в Ессентуках с бесплатным замером и контролем качества

Recommended subheadline:
> Организуем ремонт под ключ: выезд, замер, смета, подбор мастеров, контроль работ и поэтапная сдача объекта.

Trust bullets:
- Бесплатный выезд на замер;
- Смета до начала работ;
- Специалисты 10+ лет;
- Контроль качества на каждом этапе.

CTA buttons:
- Получить бесплатный замер;
- Рассчитать примерную стоимость.

## 3. Services
Services:
- отделка;
- ремонт квартир;
- ванная / санузел;
- электрика;
- строительство домов;
- кровля;
- кладка.

UX:
- card grid;
- main launch services visually stronger:
  - ремонт квартир / отделка;
  - ванная / санузел / плитка;
- secondary services marked as available by request / тестовое направление:
  - электрика;
  - строительство домов;
- delayed:
  - кровля;
  - кладка.

## 4. Why Trust Us
Content:
- специалисты 10+ лет;
- контроль работ;
- работа по этапам;
- гарантия цены и качества.

Design:
- 4 cards or horizontal metrics;
- simple icons;
- no exaggerated claims;
- no fake awards.

## 5. How We Work
Steps:
1. Заявка;
2. Консультация;
3. Выезд;
4. Смета;
5. Согласование;
6. Выполнение;
7. Контроль;
8. Сдача.

Design:
- mobile vertical timeline;
- desktop horizontal / bento timeline;
- emphasize user clarity.

## 6. Calculator
Fields:
- type of object;
- area;
- repair/work type;
- city;
- contact.

MVP rule:
The calculator should not promise final price. It gives a preliminary range and sends lead data to Telegram.

Possible label:
> Предварительный расчёт стоимости

Disclaimer:
> Точная стоимость зависит от состояния объекта, материалов и выбранного объёма работ. Финальная смета составляется после замера.

## 7. Lead Form
Fields:
- name;
- phone;
- city;
- work type;
- comment.

Submit destination:
- Telegram group with user and partner.

No CRM on MVP.

## 8. Contacts
Required:
- phone;
- WhatsApp;
- Telegram;
- zone: Essentuki and KMV.

Geo copy:
> Базируемся в Ессентуках. Работаем по КМВ: Ессентуки, Кисловодск, Железноводск, Пятигорск и соседние населённые пункты по согласованию.

## 9. Footer
Content:
- project name;
- services;
- contacts;
- legal disclaimer;
- privacy link if form exists.
