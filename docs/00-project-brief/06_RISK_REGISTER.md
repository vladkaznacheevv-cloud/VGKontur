# Регистр рисков проекта

## 1. Риск нехватки бюджета

Описание: рекламный тест может не окупиться сразу, а проект стартует с минимальными деньгами.

Вероятность: высокая.  
Влияние: высокое.

Меры:

- начинать с узкой рекламы;
- не запускать дорогие каналы сразу;
- считать квалифицированные заявки, а не просто лиды;
- не брать постоянные расходы;
- делать сайт своими силами/минимальным бюджетом;
- масштабировать только после первых договоров.

## 2. Риск некачественных бригад

Описание: клиент будет воспринимать ошибки бригады как ошибку проекта.

Вероятность: высокая.  
Влияние: критическое.

Меры:

- список проверенных мастеров;
- пробный объект;
- договоренности по ответственности;
- фотофиксация этапов;
- чек-листы приемки;
- резерв на переделки;
- не брать сложные объекты без надежной бригады.

## 3. Риск кассового разрыва

Описание: деньги клиента ушли на материалы/бригаду, а проект требует дополнительных затрат.

Вероятность: средняя.  
Влияние: высокое.

Меры:

- этапная оплата;
- отдельный учет по объектам;
- резерв 10–15%;
- не смешивать деньги разных объектов;
- не работать без аванса/этапного финансирования.

## 4. Риск юридических претензий

Описание: ремонт — конфликтная ниша; возможны претензии по срокам, качеству, смете.

Вероятность: средняя/высокая.  
Влияние: высокое.

Меры:

- договор;
- смета;
- акт выполненных работ;
- гарантийные условия;
- фиксация изменений сметы;
- письменные согласования;
- не обещать неподтвержденное.

## 5. Риск плохой рекламной экономики

Описание: заявки могут быть дорогими и нецелевыми.

Вероятность: средняя.  
Влияние: высокое.

Меры:

- запускать только горячие поисковые запросы;
- использовать минус-слова;
- отслеживать звонки/мессенджеры;
- отключать слабые группы;
- делать отдельные объявления по городам;
- быстро отвечать на заявки.

## 6. Риск демпинга

Описание: частные мастера могут предлагать цену ниже.

Вероятность: высокая.  
Влияние: среднее.

Меры:

- не конкурировать только ценой;
- продавать контроль, гарантию, договор, смету, отчетность;
- объяснять клиенту риски “дешевой бригады без контроля”.

## 7. Риск отсутствия кейсов

Описание: новый проект не имеет портфолио.

Вероятность: высокая.  
Влияние: среднее.

Меры:

- сначала честно использовать “примеры направлений”, а не фейковые кейсы;
- быстро собирать фото/видео первых объектов;
- просить отзывы;
- делать кейсы “до/после”;
- показывать процесс контроля, а не только красивые финальные фото.

## 8. Риск перегруза направлениями

Описание: отделка, дома, электрика, кровля, кладка — слишком много услуг для старта.

Вероятность: высокая.  
Влияние: среднее/высокое.

Меры:

- основной фокус: ремонт и отделка;
- побочные направления показывать как дополнительные;
- рекламу запускать сначала по отделке/ремонту;
- отдельные направления тестировать позже.

## 9. Риск недоверия клиента к модели делегирования

Описание: клиент может спросить “а кто конкретно будет делать?”.

Вероятность: средняя.  
Влияние: высокое.

Меры:

- не скрывать модель;
- позиционироваться как организация и контроль;
- говорить: “работают проверенные профильные мастера, а мы отвечаем за коммуникацию, смету, контроль и результат”;
- показывать ответственного прораба/координатора.

## 10. Риск обещаний по ГОСТам

Описание: фраза “контроль ГОСТов” звучит сильно, но требует реальной компетенции и чек-листов.

Вероятность: средняя.  
Влияние: высокое.

Меры:

- использовать формулировку “контроль соблюдения технологий, строительных норм и требований проекта”;
- подготовить чек-листы;
- не обещать экспертизу там, где нет подтвержденного специалиста;
- при сложных объектах подключать профильного специалиста.
