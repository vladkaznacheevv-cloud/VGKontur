# Источники проекта: строительные и отделочные работы КМВ

Дата первичной фиксации: 2026-06-04

## Назначение пакета

Этот пакет предназначен для загрузки в источники GPT-проекта и последующей работы над сайтом, рекламой, позиционированием, финансовой моделью и операционной схемой стартапа.

## Состав документов

1. `01_PROJECT_BRIEF.md` — суть проекта, гео, услуги, преимущества, ограничения.
2. `02_MARKET_AND_COMPETITOR_ANALYSIS.md` — первичный конкурентный анализ КМВ.
3. `03_DEMAND_AND_ADS_PRELIMINARY.md` — первичная оценка спроса, рекламных каналов, семантики и бюджета.
4. `04_MVP_SITE_ARCHITECTURE.md` — рекомендуемая архитектура лендинга/MVP-сайта.
5. `05_BUSINESS_MODEL_AND_FINANCIAL_MODEL.md` — бизнес-модель и приблизительная экономика первого года.
6. `06_RISK_REGISTER.md` — риски проекта и меры снижения.
7. `07_KEYWORD_MAP_INITIAL.md` — стартовая карта ключевых запросов для SEO/Яндекс Директ.
8. `08_NEXT_ACTION_PLAN.md` — следующий план действий.

## Важное ограничение

Точные частотности Яндекс Вордстат/Прогноз бюджета Яндекс Директ и Google Keyword Planner должны быть дополнительно выгружены из рекламных кабинетов. Внешний анализ в этом пакете является первичной рыночной разведкой, а не финальным медиапланом.

## Stage 1 update — Essentuki-first strategy

New strategic documents added after the geography decision:

- `09_GEO_PRIORITY_AND_ROLLOUT_STRATEGY.md` — fixed city rollout: Essentuki first, then Kislovodsk, Zheleznovodsk, Pyatigorsk later.
- `10_ESSENTUKI_MVP_POSITIONING_AND_SITE_SPEC.md` — MVP landing positioning and page structure for Essentuki.
- `11_STAGE_1_ACTION_PLAN_ESSENTUKI.md` — practical launch checklist for the Essentuki validation stage.

Strategic note: the project should not start as a generic KMW-wide website. The first commercially focused version should be built around Essentuki, because this is the operational base and the best city for low-budget validation.
