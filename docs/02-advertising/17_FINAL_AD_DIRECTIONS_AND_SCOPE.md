# Stage 3 — Финальная структура рекламных направлений

## Общий фокус проекта

Проект позиционируется шире, чем один город:

**Строительство и ремонт по КМВ.**

Однако первый рекламный и операционный запуск выполняется через базовый город:

**Ессентуки.**

Это нужно, чтобы не распылять минимальный бюджет, быстрее выезжать на замеры, контролировать объекты и получить первые кейсы.

## Гео-стратегия

1. Ессентуки — основной старт и база.
2. Кисловодск — следующее масштабирование после проверки Ессентуков.
3. Железноводск — следующий этап после Кисловодска.
4. Пятигорск — позже, так как там заметнее конкуренция.

## Принятый главный оффер

**Ремонт и отделочные работы в Ессентуках с бесплатным замером, понятной сметой и контролем качества на каждом этапе.**

## Финальная структура направлений рекламы

| Направление | Статус | Комментарий |
|---|---|---|
| Ремонт квартир / отделка | Основной запуск | Главный коммерческий блок первого запуска |
| Ремонт ванной / санузла / плитка | Основной запуск | Узкое направление с потенциально хорошей конверсией |
| Электрика | Малый тест | Запускать осторожно, отдельным небольшим бюджетом |
| Строительство домов | Осторожный тест | Высокий чек, но длинный цикл сделки и выше риск нецелевых обращений |
| Кровля | Позже | Не включать в первый запуск, чтобы не распылять бюджет |
| Каменная кладка | Позже | Оставить как дополнительное направление для масштабирования |

## Что не делаем в первом запуске

- Не запускаем весь КМВ одной кампанией.
- Не начинаем с Пятигорска.
- Не тратим бюджет равномерно на все услуги.
- Не запускаем кровлю и каменную кладку в первом рекламном этапе.
- Не делаем сложную CRM на старте.
- Не строим сайт вокруг всех направлений одинаково — первый экран и рекламный акцент должны быть под ремонт/отделку в Ессентуках.

## Рекламный бюджет

Рабочая вилка первого месяца:

**30 000–50 000 ₽**

Предпочтительный подход:

- начать с осторожной ставки;
- смотреть заявки через 5–7 дней;
- усиливать те группы, которые дают нормальные обращения;
- отключать или урезать группы с мусорным трафиком.

## Предварительное распределение бюджета

| Направление | Бюджет на первый месяц |
|---|---:|
| Ремонт квартир / отделка | 10 000–15 000 ₽ |
| Ванная / санузел / плитка | 10 000–15 000 ₽ |
| Электрика | 3 000–5 000 ₽ |
| Строительство домов | 5 000–8 000 ₽ |
| Резерв на усиление рабочих групп | 5 000–7 000 ₽ |

Кровля и каменная кладка в первый запуск не включаются.

## KPI первого месяца

| Метрика | Рабочий ориентир |
|---|---:|
| Клики | 250–600 |
| Заявки | 8–20 |
| Стоимость заявки | до 3 000–5 000 ₽ |
| Выезды на замер | 3–7 |
| Договоры | 1–3 |
| Конверсия сайта в заявку | от 3% |
| Конверсия заявки в замер | от 30% |
| Конверсия замера в договор | от 20–30% |

Главная метрика — не количество заявок, а стоимость нормального замера и договора.
