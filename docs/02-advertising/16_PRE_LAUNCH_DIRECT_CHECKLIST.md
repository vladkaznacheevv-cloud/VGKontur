# 16. Pre-Launch Yandex Direct Checklist

Дата фиксации: 2026-06-04

---

## 1. Что нужно сделать до запуска рекламы

### Сайт

- Сайт открывается быстро на мобильном.
- Первый экран сразу говорит: ремонт/отделка в Ессентуках.
- Есть CTA: “Получить бесплатный замер”.
- Есть WhatsApp.
- Есть телефон.
- Есть форма заявки.
- Есть онлайн-калькулятор или быстрый расчёт.
- Есть блок “Как работаем”.
- Есть блок контроля качества.
- Есть юридически аккуратные формулировки без невыполнимых обещаний.

### Аналитика

- Установлена Яндекс Метрика.
- Настроены цели:
  - phone_click;
  - whatsapp_click;
  - lead_form_submit;
  - calculator_submit;
  - measurement_request.
- Проверены UTM-метки.
- Создана таблица заявок.

### Реклама

- Кампании разделены по направлениям.
- Указан регион Ессентуки.
- Добавлены минус-слова.
- РСЯ отключена на первом запуске или вынесена отдельно.
- Объявления содержат город.
- Объявления ведут на релевантные блоки сайта.
- Дневной лимит ограничен.
- Авторасширение/автотаргетинг контролируется аккуратно.

---

## 2. Ручная проверка в Яндекс Директ

Внутри Яндекс Директ нужно вручную сделать:

1. Открыть “Прогноз бюджета”.
2. Выбрать регион: Ессентуки.
3. Вставить группы ключей отдельно:
   - ремонт квартир;
   - отделочные работы;
   - ванная/санузел;
   - электрика;
   - строительство/кровля как тест.
4. Добавить минус-слова.
5. Посмотреть прогноз:
   - показы;
   - клики;
   - CTR;
   - средняя списываемая цена;
   - месячный бюджет.
6. Повторить расчёт для более широкого региона с фразами, где явно есть “Ессентуки”.
7. Сохранить выгрузку или скриншоты в источники проекта.

---

## 3. Что нужно получить из прогноза

После ручной проверки должны быть зафиксированы:

- прогноз спроса по Ессентукам;
- какие группы имеют объём;
- какие фразы слишком дорогие;
- какие фразы слишком низкочастотные;
- реальная вилка бюджета;
- можно ли уложиться в 30–50 тыс. ₽;
- нужно ли расширять гео до КМВ с гео-фразами.

---

## 4. Решения после прогноза

Если объёма по Ессентукам достаточно:

- запускаем только Ессентуки;
- бюджет 30–50 тыс. ₽;
- сайт делаем Essentuki-first.

Если объёма мало:

- оставляем сайт Essentuki-first;
- в рекламе добавляем КМВ/Ставропольский край только с явными фразами “Ессентуки”, “КМВ”, “рядом”;
- готовим расширение на Кисловодск.

Если клики слишком дорогие:

- режем общие фразы;
- усиливаем низкочастотные запросы;
- делаем отдельные объявления под отделку/ванную;
- временно откладываем строительство домов.

---

## 5. Финальное решение перед запуском

Перед запуском должны быть согласованы:

1. Бюджет на месяц.
2. Дневной лимит.
3. Список кампаний.
4. Список ключевых фраз.
5. Список минус-слов.
6. Структура сайта.
7. Тексты первого экрана.
8. Форма заявки.
9. Ответственный за обработку заявок.
10. Скрипт первого звонка/WhatsApp.
